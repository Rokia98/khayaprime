## Patterns à appliquer

### Bouton accessible

❌

```tsx
<div onClick={onClick}>Click</div>
```

✅

```tsx
<button
  onClick={onClick}
  className="px-4 py-2 bg-secondary text-white rounded-lg focus:outline-none focus:ring-2 focus:ring-secondary"
>
  Click
</button>
```

---

### Responsive correct

❌

```tsx
<div className="flex">
```

✅

```tsx
<div className="flex flex-col md:flex-row">
```

---

### Contraste OK

❌

```tsx
text-gray-300
```

✅

```tsx
text-primary
```

---

### Card propre

✅

```tsx
<div className="p-4 rounded-2xl shadow-sm bg-white hover:shadow-md transition">
```

---

## Optimisations automatiques

Toujours :

* ajouter `transition` (200–300ms)
* ajouter `hover` state
* ajouter `focus:ring`
* éviter `div clickable`
* corriger spacing (p-4, gap-4)
* éviter les classes inutiles
